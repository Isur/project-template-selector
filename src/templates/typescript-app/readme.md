# Typescript starter app.

## Usage
Install dependencies:
```bash
npm install
```

Testing:
```bash
npm run test
```
```bash
npm run test:watch
```
Linter:
```bash
npm run lint
```
```bash
npm run lint:fix
```

Build:
```bash
npm run build
```
Start
```bash
npm run start
```
Start with reload on changes
```bash
npm run dev
```
